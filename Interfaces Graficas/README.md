<br/>

- <h3>Ventana click visitas guiadas :</h3> Digitar datos básicos --> Nombre completo, ID y Celular. <br/> Botón 'OK'

- <h3>Ventana click Estudiantes inscritos :</h3> Digitar ID. <br/> Botón 'OK'
